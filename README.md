# 🎮 سنگ، کاغذ، قیچی

یک بازی سنگ کاغذ قیچی ساده و جذاب که توی ترمینال اجرا میشه. این بازی رو برای تمرین پایتون نوشتم و چندتا قابلیت جالب بهش اضافه کردم.

## چی توش هست؟

- بازی در ۵ دور انجام میشه (میتونی تعداد دور رو عوض کنی)
- ۳ سطح سختی داره: آسون، متوسط و سخت
- تاریخچه حرکات رو نگه میداره (آخرین ۵ حرکت)
- از ایموجی و رنگ برای زیباتر شدن استفاده کرده
- کامپیوتر تو سطح سخت واقعاً باهوشه!

## چطور اجرا کنم؟

فقط کافیه پایتون ۳ رو داشته باشی و فایل رو اجرا کنی:

```bash
python game.py
